1.Создать HTML-страницу для отображения/редактирования текста. При открытии страницы текст отображается с помощью тега div. При нажатии Ctrl + V, вместо div появляется textarea с тем же текстом, который теперь можно редактировать. При нажатии Ctrl + M , вместо textarea появляется div с уже измененным текстом. Не забудьте выключить поведение по умолчанию для этих сочетаний клавиш.
